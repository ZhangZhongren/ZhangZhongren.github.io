# 动态路由配合 json-server

  1. 新建 db.json
 {
  "list": [
    {
      "name": "zzr",
      "age": "25",
      "birthday": "1995、08、02"
    }
  ]
}
  1. 新增 router.js
route('/api/(.*)')
.rewrite('/{0}')
.to.jsonServer();
  1. 启动
svrx --route router.js -p json-server
  1. 访问
http://localhost:8000/api/list
